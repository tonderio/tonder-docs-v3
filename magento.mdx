The Tonder Plugin for Magento 2.0 is a robust integration solution designed to seamlessly connect your Magento-based e-commerce platform with Tonder's advanced payment orchestration platform. 

https://commercemarketplace.adobe.com/tonder-module-payment.html#description

<Frame>![](/images/magento1.png)</Frame>


### Instructions

- **Step 1 Access to the Terminal:**
Open a terminal window on your server where Magento is installed.Access the root directory of your Magento installation using the command


```
cd /ruta/a/tu/instalacion/magento

```

- **Step 2 Plugin Download:**

Use Composer to install the plugin. Run the following command to download the plugin:
```
composer require tonder/module-payment dev-m2.4.2
```

- **Step 3 Enable the Plugin:**
Once the download is complete, run the following command to enable the plugin:
```
bin/magento module:enable Tonder_Payment --clear-static-content
```

- **Step 4 System Update:**
After enabling the plugin, run the following command to perform the necessary updates on the system:

```
bin/magento setup:upgrade
```

- **Step 5 Run the command:** Run the following command to compile the dependency injection classes:

```
bin/magento setup:di:compile
```


- **Step 6 Cleaning the Cache:**
To ensure that your changes are reflected correctly, run the following command to clear the cache:

```
php bin/magento cache:flush
```

- **Step 7 Tonder Payment Plugin Configuration:**
Access the Magento administration console to configure the plugin. 
Once inside, go to Stores -> Settings -> Sales -> Payment Methods and search for Tonder Payment Gateway
Click and select in **'Enable'** the --yes-- option

<Frame>![](/images/magento2.png)</Frame>

- **Step 8 Change the name:** Change the name 'Tonder' this name will appear in the magento checkout and add
**“Pagar con tarjeta de crédito/débito”**

<Frame>![](/images/magento3.png)</Frame>

- **Step 9 Choose the server:** Choose the server you want. 

<Frame>![](/images/magento3.png)</Frame>

- **Step 10 Add your keys:** Enter your Merchant ID and Token (Provided by Tonder in [Developers](./developers) section)

<Frame>![](/images/magento6.png)</Frame>

-  Choose the Recommended Payment Action **“Authorize and Capture”**. In order to complete the complete orders and click the Save Config. button.

<Frame>![](/images/magento5.png)</Frame>

- **Finally, delete the cache and perform your necessary tests.**
