This reference provides the complete list of Mexican financial institution codes used in SPEI transactions and withdrawals. These codes are assigned by Banco de México (Banxico) and are required for processing interbank transfers.

<Info>
This list contains **86 institutions** as registered SPEI participants. Institution codes are used in the `institution` field when creating withdrawal requests.
</Info>

## Bank Account Types

There are different account types used in Mexican banking:

| Code | Type               | Description             |
|------|--------------------| ----------------------- |
| 1    | Checking Account   | Cuenta de cheques       |
| 2    | Savings Account    | Cuenta de ahorros       |
| 3    | Credit Account     | Cuenta de crédito       |
| 40   | CLABE Account      | Cuenta CLABE (for SPEI) |
| 10   | Investment Account | Cuenta de inversión     |

For SPEI withdrawals, the account type is usually picked up automatically from the CLABE format. However, for some operations, you might need to specify it yourself.

## Example: Beneficiary Object

This example shows a complete beneficiary object for a SPEI withdrawal request:

```json
{
  "beneficiary": {
    "account": "012345678901234567",
    "account_type": "40",
    "name": "Roberto Martínez García",
    "rfc": "MAGR850920XY1",
    "institution": "40012"
  }
}
```

| Field | Type | Description |
|-------|------|-------------|
| `account` | string | The 18-digit CLABE number identifying the recipient's bank account. |
| `account_type` | string | Code "40" indicates this is a CLABE account suitable for SPEI transfers. |
| `name` | string | Legal name of the account holder, must match bank records. |
| `rfc` | string | Mexican tax ID for the account holder (individual or business). |
| `institution` | string | Institution code from the tables above. |

<Tip>
The institution code corresponds to the first 3 digits of the CLABE number. For example, a CLABE starting with `012` belongs to BBVA Mexico (code `40012`).
</Tip>
