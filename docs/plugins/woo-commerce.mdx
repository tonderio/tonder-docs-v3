## Installation

To start using Tonder's plugin with WooCommerce on your website, you need to perform the following steps:

1. Have WooCommerce integrated into your site.
2. [Install the Tonder plugin](https://wordpress.org/plugins/tonder-payments-gateway/).
3. Navigate to **WooCommerce** > **Settings** > **Payments**.
4. Activate Tonder's plugin.
5. Click on **Tonder Pagos**.
6. Fill in your API Credentials and Tonder ID.

<Note>Your Credentials can be found in the **[Developers](https://stage.tonder.io/dashboard/developers/)** tab of Tonder's dashboard.</Note>
