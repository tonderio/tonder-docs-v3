---
title: "Make a Payment (Ionic)"
description: "Learn how to accept a payment in your Ionic app using Tonder's SDK."
---

This guide walks you through processing a payment using the **Full SDK** (`InlineCheckout`) for Ionic.

## Prerequisites

-   You have successfully [installed and configured](/sdk-integration/mobile/ionic/installation) the Tonder Ionic SDK.
-   You have a checkout page with an empty `div` element to contain the SDK's UI.

```html
<div id="tonder-checkout"></div>
<button id="pay-button">Pay Now</button>
```

## Steps

1. Initialize the SDK
Import and create a new instance of InlineCheckout, providing your API key, return URL, and operating mode.

```javascript

import { InlineCheckout } from "@tonder.io/ionic-full-sdk";

const inlineCheckout = new InlineCheckout({
  apiKey: "YOUR_API_KEY",
  returnUrl: "YOUR_RETURN_URL",
  mode: "stage", // or "production"
});
```

2. Inject the Checkout UI
Call the injectCheckout() method to render the pre-built payment form inside your designated div.

```javascript

inlineCheckout.injectCheckout();
```

3. Process the Payment
Add an event listener to your payment button. When clicked, call the payment() method, passing the customer and cart data.

```javascript

document.getElementById('your-pay-button').addEventListener('click', async () => {
  const paymentData = {
    customer: { firstName: "Juan", email: "juan.hernandez@mail.com", /* ... */ },
    cart: { total: 399, items: [ /* ... */ ] },
    currency: "mxn"
  };

  try {
    const response = await inlineCheckout.payment(paymentData);
    console.log('Payment successful:', response);
  } catch (error) {
    console.error('Payment failed:', error);
  }
});
```

4. Handle 3DS Verification
If a 3D Secure challenge is required, the SDK will handle the redirection. After the user returns to your returnUrl, call verify3dsTransaction() to get the final transaction status.

```javascript

// On your return page
inlineCheckout.verify3dsTransaction().then(response => {
  if (response.transaction_status === 'Success') {
    alert('3DS Transaction verified.');
  } else {
    alert('3DS Transaction Failed');
  }
});
```
